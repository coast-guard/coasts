Вы настраиваете навык Coasts для T3 Code в этом проекте. Выполняйте все
команды из корня проекта.

## Шаг 1: Проверьте наличие Coast CLI

Выполните эту команду:

  coast --version

Если команда `coast` не найдена, остановитесь здесь и сообщите пользователю:

  "Coast CLI не установлен. Сначала установите его: https://coasts.dev/docs/getting-started"

Не продолжайте, пока CLI не станет доступен.

## Шаг 2: Получите содержимое навыка

Выполните эту команду:

  coast skills-prompt

Вывод состоит из двух частей:

- **Правила Coast Runtime** — всё от начала до строки, которая начинается с `---` (но не включая её)
- **Навык Coasts** — всё, начиная с блока frontmatter `---` и далее (включая строки `---` и всё после них)

Сохраните обе части для следующих шагов.

## Шаг 3: Спросите пользователя

T3 Code поддерживает несколько провайдеров. Спросите пользователя: каких провайдеров вы используете
в T3 Code?

- **Codex** (OpenAI)
- **Claude** (Anthropic)
- **Оба**
- **Другое** — если пользователь укажет другого провайдера, выполните
  `coast docs --path SKILLS_FOR_HOST_AGENTS.md` и
  `coast docs --path harnesses/README.md`, чтобы определить, где этот провайдер
  ожидает инструкции проекта и навыки, затем следуйте тому же шаблону

## Шаг 4: Разместите файлы

Настройка T3 Code всегда выполняется на уровне проекта. Если целевой файл уже существует, добавьте
раздел Coast Runtime вместо перезаписи — но сначала проверьте, присутствует ли уже
раздел `# Coast Runtime`, и если да, пропустите.

**Если пользователь выбрал Codex (или оба):**
- Добавьте правила Coast Runtime в `AGENTS.md` в корне проекта
- Запишите навык Coasts в `.agents/skills/coasts/SKILL.md` (уровень проекта)
- Также запишите навык Coasts в `~/.codex/skills/coasts/SKILL.md` (глобальный
  fallback — T3 Code, возможно, ещё не сканирует `.agents/skills/` на уровне проекта)
- Создайте `.agents/skills/coasts/agents/openai.yaml` со следующим содержимым:

interface:
  display_name: "Coasts"
  short_description: "Inspect, assign, and open Coasts for this repo"
  default_prompt: "Use this skill when the user wants help finding, assigning, or opening a Coast."

policy:
  allow_implicit_invocation: false

**Если пользователь выбрал Claude (или оба):**
- Добавьте правила Coast Runtime в `CLAUDE.md` в корне проекта
- Запишите навык Coasts в `.claude/skills/coasts/SKILL.md` (уровень проекта)
- Также запишите навык Coasts в `~/.claude/skills/coasts/SKILL.md` (глобальный
  fallback — T3 Code, возможно, ещё не сканирует `.claude/skills/` на уровне проекта надёжно)

**Примечание:** T3 Code, возможно, пока не загружает навыки уровня проекта из `.agents/skills/` или
`.claude/skills/`. Правила в `AGENTS.md` и `CLAUDE.md` всё равно применяются
в каждой задаче. Глобальный fallback `~/.codex/skills/coasts/` гарантирует,
что провайдер Codex сможет найти навык.

## Шаг 5: Обновите Coastfile

Прочитайте `Coastfile` в корне проекта. Посмотрите на поле `worktree_dir` в
разделе `[coast]`. Также прочитайте поле `name` — оно понадобится вам, чтобы построить
путь worktree.

Каталог worktree для T3 Code — это `~/.t3/worktrees/<name>`, где `<name>` —
значение поля `name` в Coastfile.

Если этот путь **ещё не** указан в `worktree_dir`:

- Если `worktree_dir` — это одна строка, преобразуйте её в массив и добавьте
  путь T3 Code. Например, `worktree_dir = ".worktrees"` становится
  `worktree_dir = [".worktrees", "~/.t3/worktrees/my-app"]`.
- Если `worktree_dir` уже является массивом, добавьте в него путь T3 Code.
- Если `worktree_dir` вообще отсутствует, добавьте
  `worktree_dir = [".worktrees", "~/.t3/worktrees/<name>"]`, используя
  фактическое имя проекта.

Это внешний путь, поэтому, если экземпляр Coast уже запущен для этого
проекта, его нужно пересоздать с помощью `coast run`, чтобы новый bind mount
вступил в силу. Сообщите об этом пользователю.

## Шаг 6: Подтвердите

После размещения файлов покажите пользователю сводку того, что было создано и где.
Объясните, что глобальная копия `~/.codex/skills/coasts/` — это обходной путь из-за того, что
T3 Code пока не загружает навыки уровня проекта.

Сообщите пользователю: **Перезапустите T3 Code**, чтобы изменения навыка и правил
вступили в силу.
