Вы настраиваете навык Coasts для Shep в этом проекте. Выполняйте все
команды из корня проекта.

## Шаг 1: Проверьте наличие Coast CLI

Выполните эту команду:

  coast --version

Если команда `coast` не найдена, остановитесь здесь и сообщите пользователю:

  "Coast CLI не установлен. Сначала установите его: https://coasts.dev/docs/getting-started"

Не продолжайте, пока CLI не станет доступен.

## Шаг 2: Получите содержимое навыка

Выполните эту команду:

  coast skills-prompt

Вывод состоит из двух частей:

- **Правила Coast Runtime** — всё от начала до строки, начинающейся с `---` (но не включая её)
- **Навык Coasts** — всё, начиная с блока frontmatter `---` и далее (включая строки `---` и всё после них)

Сохраните обе части для следующих шагов ниже.

## Шаг 3: Разместите файлы

Shep оборачивает Claude Code, поэтому используйте файловую структуру Claude Code. Если целевой файл
уже существует, добавьте раздел Coast Runtime в конец вместо перезаписи — но
сначала проверьте, присутствует ли уже раздел `# Coast Runtime`, и если
да, пропустите этот шаг.

- Добавьте правила Coast Runtime в `CLAUDE.md` в корне проекта
- Запишите навык Coasts в `.agents/skills/coasts/SKILL.md` (или
  `.claude/skills/coasts/SKILL.md`, если проект уже использует такую структуру)

## Шаг 4: Обновите Coastfile

Прочитайте `Coastfile` в корне проекта. Посмотрите на поле `worktree_dir` в
разделе `[coast]`.

Если `~/.shep/repos/*/wt` **ещё не** указан в `worktree_dir`:

- Если `worktree_dir` — это одна строка, преобразуйте её в массив и добавьте
  `~/.shep/repos/*/wt`. Например, `worktree_dir = ".worktrees"` становится
  `worktree_dir = [".worktrees", "~/.shep/repos/*/wt"]`.
- Если `worktree_dir` уже является массивом, добавьте в него `~/.shep/repos/*/wt`.
- Если `worktree_dir` вообще отсутствует, добавьте
  `worktree_dir = [".worktrees", "~/.shep/repos/*/wt"]`.

`*` — это glob-шаблон, который соответствует директории хеша для каждого репозитория. Coasts
разворачивает его во время выполнения.

Это внешний путь, поэтому если для этого проекта уже запущен экземпляр Coast,
его необходимо пересоздать с помощью `coast run`, чтобы новый bind mount вступил
в силу. Сообщите об этом пользователю.

## Шаг 5: Подтвердите

После размещения файлов покажите пользователю сводку о том, что было создано и где.

Сообщите пользователю: **Закройте и заново откройте ваш редактор**, чтобы изменения
навыка и `CLAUDE.md` вступили в силу.
