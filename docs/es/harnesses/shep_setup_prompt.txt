Estás configurando la skill de Coasts para Shep en este proyecto. Ejecuta todos
los comandos desde la raíz del proyecto.

## Paso 1: Verificar Coast CLI

Ejecuta este comando:

  coast --version

Si no se encuentra el comando `coast`, detente aquí y dile al usuario:

  "The Coast CLI is not installed. Install it first: https://coasts.dev/docs/getting-started"

No continúes hasta que el CLI esté disponible.

## Paso 2: Obtener el contenido de la skill

Ejecuta este comando:

  coast skills-prompt

La salida tiene dos partes:

- **Reglas de Coast Runtime** — todo desde el inicio hasta (pero sin incluir) la línea que comienza con `---`
- **Skill de Coasts** — todo desde el bloque de frontmatter `---` en adelante (incluyendo las líneas `---` y todo lo que sigue después)

Guarda ambas partes para los pasos a continuación.

## Paso 3: Colocar archivos

Shep envuelve Claude Code, así que usa la estructura de archivos de Claude Code. Si un archivo de destino
ya existe, agrega la sección de Coast Runtime en lugar de sobrescribirla, pero
primero verifica si ya está presente una sección `# Coast Runtime` y omítela si
es así.

- Agrega las reglas de Coast Runtime a `CLAUDE.md` en la raíz del proyecto
- Escribe la skill de Coasts en `.agents/skills/coasts/SKILL.md` (o
  `.claude/skills/coasts/SKILL.md` si el proyecto ya usa esa estructura)

## Paso 4: Actualizar el Coastfile

Lee el `Coastfile` en la raíz del proyecto. Observa el campo `worktree_dir` en la
sección `[coast]`.

Si `~/.shep/repos/*/wt` **aún no** está incluido en `worktree_dir`:

- Si `worktree_dir` es una sola cadena, conviértelo en un arreglo y agrega
  `~/.shep/repos/*/wt`. Por ejemplo, `worktree_dir = ".worktrees"` se convierte en
  `worktree_dir = [".worktrees", "~/.shep/repos/*/wt"]`.
- Si `worktree_dir` ya es un arreglo, agrega `~/.shep/repos/*/wt`.
- Si `worktree_dir` no está presente en absoluto, agrega
  `worktree_dir = [".worktrees", "~/.shep/repos/*/wt"]`.

El `*` es un patrón glob que coincide con el directorio hash por repositorio. Coasts
lo expande en tiempo de ejecución.

Esta es una ruta externa, así que si ya se está ejecutando una instancia de Coast para este
proyecto, debe recrearse con `coast run` para que el nuevo bind mount surta
efecto. Díselo al usuario.

## Paso 5: Confirmar

Después de colocar los archivos, muestra al usuario un resumen de lo que se creó y dónde.

Dile al usuario: **Cierra y vuelve a abrir tu editor** para que los cambios de la skill y de CLAUDE.md
surtan efecto.
